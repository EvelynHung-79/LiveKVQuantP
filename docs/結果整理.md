## 關鍵發現
- **warmup=false 幾乎無影響**（0.2938 vs 0.2957），可考慮移除 warmup 以簡化。但為了保持與原始實驗一致，最終版本仍保留 warmup。
- **outlier isolation 有明顯效益**：移除後 -0.0224（-7.6%），但換來 -12% latency
- **ema_minmax（asymmetric）表現差於 symmetric**（0.2725 vs 0.2957），不建議採用
- **chunk_size=512 是最佳 score**；chunk_size=4096 有最低 latency（9.7s）且 score 接近（0.2935），若 latency 優先可考慮
- **outlier_ratio=1% 是最佳**：0.5% 不足以捕捉 outlier（-0.0147），5% 過多反而增加 memory 且 score 下降（-0.0132）
- **quant_start_layer=0（全層量化）score 下降**（0.2846 vs 0.2957, -3.8%），但 peak memory 最低（26,245 MB）
- **ema_alpha=0.1 是最佳**（0.2957），alpha 越大 score 越低，較慢的 EMA 更新保留更多歷史統計，有助穩定量化
- **clip_factor 隨增大而 score 單調下降**：4.0 最高（0.2957），7.0 最低（0.2795），過大的 clip range 降低量化精度
